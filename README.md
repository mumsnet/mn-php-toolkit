# mn-toolkit

This repo houses the source for our composer package `mn-toolkit`.  This package contains the following features:

* Feature toggles
* Local file cache
* Logger class for this package
* Globals frontend
* JWT
* Transactional emails
* Source IP detection
* Graylog (site actions)
* Correlation ID
* User sessions (SSO) for lambda
* User sessions (SSO) for laravel


## Install

Via Composer

``` bash
$ composer require mumsnet/mn-toolkit
```
